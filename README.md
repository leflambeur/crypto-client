crypto.com go client for learning go concepts

```
./crypto-client                                                                                                                                                                                                                                                                                                                                            (main)
NAME:
   crypto-client - Interact with crypto.com API

USAGE:
    [global options] command [command options] [arguments...]

COMMANDS:
   get-instruments, i  Get instruments from crypto.com
   get-book, book      Get book info for Instrument from crypto.com
   help, h             Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h  show help
```
